Detailed Description

template<typename K>
requires ((requires(const K &a, const K &b) { { a < b } -> std::convertible_to<bool>; }) && std::copy_constructible<K>)
struct jh::avl::avl_node< K, jh::typed::monostate >

Node element for set-style contiguous AVL trees.

Template Parameters
Kkey type (strictly ordered)

Specialization used by ordered-set semantics. Stores only the key and maintains identical AVL linkage fields (parent/left/right/height). Provides no mapped value; the node's identity is its key. Layout and balancing behavior remain consistent with the general node template.
