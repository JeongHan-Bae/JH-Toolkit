Detailed Description

template<pod_like T>
struct jh::pod::span< T >

Non-owning typed view over a contiguous memory block.

Behaves like a stripped-down std::span, but remains fully POD (T* + uint64_t).

Differences from std::span:

  • No bounds-checked .at()
  • No dynamic extent; always runtime-sized
  • No interop with iterator-only containers

Usage Model:

  • Provides indexing (operator[])
  • Range iteration via begin()/end()
  • Slicing via sub(), first(), last()

TElement type. Must satisfy pod_like.

bool jh::pod::span< T >::operator== ( const span< T > & rhs) const
constexprdefault

Equality comparison between two spans.

Two spans are considered equal if they reference the same sequence object (i.e. identical pointer and identical length). This does not compare the element values.

Note
Implemented with = default, which checks data and len. If you need value-wise comparison, use algorithms such as std::equal(lhs.begin(), lhs.end(), rhs.begin()).
